Getting Started with Brain Simulator II

Overview:

The Brain Simulator II contains innovative ideas toward creating Artificial General Intelligence (AGI). This is an ongoing Research Project. So rather than waiting for a "finished" version, we present the current state of development. We welcome suggestions, comments, and participation.

One valuable feature of Brain Simulator II is that it allows experimentation with many different algorithms. The source code download includes many features which are previews of future abilities, experiments which have been superseded, and ideas which are still under development. Our development technique follows the AGILE development process of creating solutions for the simplest cases first directed at long-term objectives. The objectives are detailed in the book, Will Computers Revolt? which you can download FREE here.

Getting Started:

1. BrainSim networks are like documents—you can load them, edit them, save them, etc.  Many display “Notes” which display automatically explaining each network and how to use it. Many have related YouTube videos. You may need to drag some windows to compensate for differences in screen resolution.

2. Try these sample networks to explore various features of the Brain Simulator II. You can open these sample network files from the File Menu.

  • BasicNeurons—illustrates the simplest neuron models.
  • CameraTest---inputs the camera on your computer to neuron signals.
  • SpeechTest—uses speech-recognition and synthesis.
  • SimVision—shows Sallie's simulated environment and vision.
  • Imagination—shows how Sallie's internal mental model can be used for imagination.
  • BabyTalk—learns to speak by trial-and-error learning.
  • Maze—shows navigation like a 3-year-old.
  • Sallie—is an end-to-end AGI model using hearing, vision, and knowledge to learn the meanings of a few words.
  • 3DSim—demonstrates the 3D simulator under development.
  • NeuralGraph—demonstrates how a mathematical graph can be implemented in neurons controlled by a module.
  • ObjectMotion—demonstrates how Sallie can move objects in her environment.
